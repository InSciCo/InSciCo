Management Scripts Setup

Prerequisites: AWS Mgmt Account Setup complete
Estimated time: 2 minutes, Cost: none
Perform this step on the Management Workstation. It is fine if this workstation will also be used as a Developer's Workstation.

We provide Microsoft PowerShell scripts to automate various steps in the setup and management of your AWS Organization. You will not need to learn PowerShell scripting to use these scripts. If you have Windows 10, PowerShell is already installed. If you are using MacOS or some variant of Linux, PowerShell is available.

The scripts call the AWS CLI to configure your AWS Organization. They are easy to read and heavily commented. We encourage you to read through the scripts and follow the documentation links provided in the scripts to learn more about the powerful commands in the AWS CLI.

  1. newtab}}> Install PowerShell

  2. Install Git Client - if you do not already have it.

  3. Clone the LazyStackSMF git repository: 
    git clone https://github.com/InSciCo/LazyStackSMF.git
    This creates a repository folder called LazyStackSMF.
  4. Change the name of the repository folder to "OrgMgmt"; replacing "Org" with your OrgCode. This folder will become your reference folder for management related actions on your organization's AWS Organization.

Step Summary

In this step we installed PowerShell and downloaded LazyStack management scripts into an OrgMgmt folder.
