Management Scripts Setup

Prerequisites: AWS Mgmt Account Setup complete
Estimated time: 2 minutes, Cost: none
Perform this step on the Management Workstation.

We provide PowerShell scripts to automate various steps in the setup and management of your AWS Organization. You will not need to learn PowerShell scripting to use these scripts.

The scripts call the AWS CLI to configure your AWS Organization. They are easy to read and heavily commented. We encourage you to read through the scripts and follow the documentation links provided in the scripts to learn more about the powerful commands in the AWS CLI.

  1. newtab}}> Install PowerShell

  2. Download the following scripts and move them into your OrgMgmt folder (replace "Org" with your OrgCode):
    • CreateOrganization.ps1
    • CreateDevAccount.ps1
    • CreateReviewAccount.ps1
    • CreateProdAccount.ps1
    • CreateCodePipeline.ps1

Step Summary

In this step we installed PowerShell and downloaded LazyStack management scripts.
