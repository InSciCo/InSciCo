AWS CI/CD Environment Overview

Setting up team collaboration with Continuous Integration / Continuous Deployment can be easily accomplished using GitHub and AWS CodeBuild.

  • Continuous Integration automates building and testing a codebase changes prior to submitting those changes for approval.
  • Continuous Deployment automates deploying "stacks" into Test and Production accounts.

GitHub, Feature Branch Workflow and AWS CodeBuild with Visual Studio IDE

To allow this guide to be immediately practical and easily implemented, we assume you will use Git for version control, a GitHub account to host your Git shared repository, and a simple Feature Branch Workflow with GitHub Pull Requests to manage team collaboration. Focusing on this popular combination allows us to offer concrete and very specific usage instructions along with AWS CodeBuild projects to implement CI/CD.
