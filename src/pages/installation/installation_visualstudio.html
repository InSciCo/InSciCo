Visual Studio Extension

Prerequisites

  • Visual Studio 2019 for Windows
      Required Workloads:
    • ASP.NET and web development
    • .NET desktop development
    • Mobile development with .NET
    • .NET Core cross-platform development
  • AWS Account
  • AWS CLI
  • AWS Toolkit for Visual Studio 2017 and 2019

Quick Start

  1. Install Visual Studio 2019 for Windows -- Community Edition works fine
  2. Create an AWS Account
  3. Install AWS CLI
  4. Install the AWS Toolkit for Visual Studio from the visual Studio Marketplace.
  5. Install LazyStack Visual Studio Extension from the Visual Studio Marketplace.
  6. Create a new project using the LazyStackWebApi Template.
  7. Author an OpenApi specification for your application stack and place it in the solution folder.
  8. Use the "Tools/Invoke LazyStack - Generate Projects" menu option. Application projects will be generated.
  9. Place any required LazyStack Directives in the LazyStack.yaml configuration file.
  10. Implement service-side application code for the the generated Controller Interfaces.
  11. Add a LazyStackAuth project to your solution
  12. Create a client application that references the generated application ClientSDK and LazyStackAuth project.
  13. Use the "Tools/Invoke LazyStack - Generate Projects" menu option. Application projects will be generated.
  14. Right-click on the generated serverless.template file in solution folder and select Publish to AWS Lambda
  15. Use the "Tools/Invoke LazyStack - Generate Stacks\Dev\AwsSettings.json" menu option. a AwsSettings file will be generated.
  16. Update client application to use the AwsSettings file as an embedded resource.
  17. Update your client application to initialize and use the LazyStackAuth and ClientSDK projects.
  18. Test your service-side application code using the local WebApi and/or AWS Stack.

Our Tutorial takes you through all these steps in detail. Or, you can just clone the LazyStackPetStoreTutorial repo from GitHub and review the projects in it.
