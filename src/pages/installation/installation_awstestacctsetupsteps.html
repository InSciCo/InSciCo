AWS Test Account Setup

Prerequisites:
- Test OU Setup Complete
Estimated Time: 2 minutes
Estimated Cost: none
Perform this step on the Management Workstation.
  1. Open a PowerShell Terminal.
  2. CD into the OrgMgmt folder. Using your OrgCode in place of "Org".
  3. Execute the Test_Account_Create.ps1 script: 
    . .\Test_Account_Create.ps1

Step Summary

In this step we used the Test_Account_Create.ps1 script to create a Test Account under the Test Organization Unit. We added a Administrators group to the account with AdministratorAccess permissions. We added an IAM User for the account and added that user to the Administrators group. We created a AWS CLI profile allowing the Management Account IAM User to manage the Test account with the OrganizationAccountAccessRole policy.

The IAM User created for this account will be used by the CI/CD process.
