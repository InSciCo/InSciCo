LazyStack Development Tools

The LazyStack Development tools are available as a Visual Studio IDE (Windows) extension and as .Net CLI tools. The Visual Studio IDE (MacOS) may be configured to use the .Net CLI Tools. The .Net CLI tools can be used in any environment supporting .NET.

.NET CLI LazyStack Tools - Mandatory

We recommend that all development workstations have the .NET CLI LazyStack Tools installed.

Visual Studio IDE (Windows) with LazyStack Extension - Optional

We recommend that Visual Studio IDE (Windows) users use the LazyStack extension available from the Visual Studio Extension Marketplace.

Visual Studio IDE (MacOS) with .NET CLI LazyStack Tools - Optional

Visual Studio IDE (MacOS) is supported using through the External Tools feature. Unfortunately, the Visual Studio Extensions Gallery is no longer accepting submissions. But, no worries, the External Tools configuration is simple and works quite well.

Visual Code and other IDEs and Editors - Optional

Like Visual Studio IDE (MacOS), many editors and IDEs have the ability to make calls to CLI tools. If you are working in one of these editors or IDEs, review the Visual Studio IDE (MacOS) setup. This setup is likely similar to what you need.
