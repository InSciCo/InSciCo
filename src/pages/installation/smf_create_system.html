SMF Step 2 - Create System

Prerequisites:
- Create AWS Org complete
- Unique email for root user of new System Test AWS Account
- Unique email for root user of new System Production AWS Account
- OrgCode (used when organization was created)
- SystemCode prefix (system short code to use for naming accounts that are part of system)
Estimated Time: 4 minutes
Estimated Cost: none
Perform this step on the Management Workstation.

Systems are comprised of a System Test Account and a System Production Account. Systems contain one or more Serverless Stacks and these are configured in Step 3 - Configure Repo(s).

We recommend that you create an initial System called Tut (short for Tutorial) that can be used by developers going through the LazyStack tutorial.
