SMF Step 3 - Create System

Prerequisites:
- Create AWS Org complete
- Unique email for root user of new System Test AWS Account
- Unique email for root user of new System Production AWS Account
- OrgCode (used when organization was created)
- SystemCode prefix (system short code to use for naming accounts that are part of system)
Estimated Time: 4 minutes
Estimated Cost: none
Perform this step on the Management Workstation.

Systems are comprised of a System Test Account and a System Production Account. Systems contain one or more Serverless Stacks and these are created in Step 4 - Create Stack.

Note: If you do not yet have a System to define you can stop the SMF setup here and come back to this step, and the following Create Stack (Step 4), later.
