AWS Organization Setup

Prerequisites: Mgmt. Scripts Setup complete
Estimated Time: 2 minutes, Costs: none
Perform this step on the Management Workstation.

In this step we will use the CreateOrganization.ps1 script to create an AWS Organization, and the OrgDev Organizational Unit.

  1. Open a PowerShell Terminal.
  2. CD into the OrgMgmt folder. Using your OrgCode in place of "Org".
  3. Issue the following command: 
    . .\CreateOrganization.ps1
    You will be prompted for your OrgCode and confirmation of the AWS CLI Profile to use.
    Here is an example of running the script:
