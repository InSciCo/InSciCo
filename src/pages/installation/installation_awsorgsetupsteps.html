Create AWS Organization

Prerequisites: AWS Mgmt. Account Setup Complete
Estimated Time: 2 minutes
Estimated Cost: none
Perform this step on the Management Workstation.

In this step we will use the Org_Create.ps1 script to create an AWS Organization, and the OrgDev Organizational Unit.

  1. Open a PowerShell terminal.
  2. CD into the OrgMgmt folder. Replace "Org" with your OrgCode.
  3. Issue the following command: 
    . .\Org_Create.ps1
    You will be prompted for your OrgCode and confirmation of the AWS CLI Profile to use.
    Here is an example of running the script:
