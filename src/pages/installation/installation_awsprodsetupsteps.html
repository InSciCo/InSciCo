AWS Production Account Setup

Prerequisites: Organization Setup completed
Estimated Time: 2 minutes, Costs: none
Perform this step on the Management Workstation

  1. Open a PowerShell Terminal.
  2. CD into the OrgMgmt folder. Replace "Org" with your OrgCode.
  3. Execute the CreateProdAccount.ps1 script: 
    . .\CreateProdAcct.ps1

Step Summary

In this step we used the CreateProdAcct.ps1 script to create a Prod Account under the Prod Organization Unit. We added a Administrators group to the account with AdministratorAccess permissions. We added an IAM User for the account and added that user to the Administrators group. We created a AWS CLI profile allowing the Management Account IAM User to manage the Prod account with the OrganizationAccountAccessRole policy.

The IAM User created for this account will be used by the CI/CD process.

Publishing Stacks to Prod Account

Using the Management Workstation (which has a profile allowing administrative access to the Prod Account, you can publish stacks using either the AWS SAM CLI or the Visual Studio IDE (Windows) AWS Toolkit Extensions.
