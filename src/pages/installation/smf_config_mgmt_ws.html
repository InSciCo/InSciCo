SMF 1.1 Configure Management Workstation

Prerequisites: none
Estimated Time: 2 to 10 minutes
Estimated Cost: none
Perform this step on the Management Workstation.

We provide Microsoft PowerShell scripts to automate various steps in the setup and management of your AWS Organization. You will not need to learn PowerShell scripting to use these scripts. If you have Windows 10, PowerShell is already installed. If you are using MacOS or some variant of Linux, PowerShell is available to install.

The scripts call the AWS CLI to configure your AWS Organization. They are easy to read and heavily commented. We encourage you to read through the scripts and follow the documentation links provided in the scripts to learn more about the powerful commands in the AWS CLI.

In this step, we will be creating a "OrgMgmt" folder on the Management Workstation containing scripts and sensitive information. Give some thought as to where you will want this folder to reside and who should have access to it.

  1. newtab}}> Install PowerShell
     * If you are running Windows 10, PowerShell is already installed.

  2. Install a Git client
    * If you do not already have it.
    • newtab}}>GitBash for Windows - this is our favorite for Windows
    • newtab}}>Git for MacOS, Linux/Unix, Windows

  3. Open a PowerShell terminal with "Run as Administrator" and issue the following command to change the PowerShell script execution permissions: 
    Set-ExecutionPolicy RemoteSigned
    This will allow you to run PowerShell scripts on the local computer which is generally necessary for the Management Workstation.

  4. Close PowerShell. We don't need Administrator access for the next steps.

  5. Open a PoserShell terminal and CD to the folder that will contain your "OrgMgmt" folder and then issue the following command to clone the LazyStackSMF git repository: 
    git clone https://github.com/InSciCo/LazyStackSMF.git
    The clone command creates a folder called LazyStackSMF.

  6. Rename the folder "LazyStackSMF" to "OrgMgmt"; replace "Org" with your OrgCode. This folder will become your reference folder for management related actions on your AWS Organization.

Step Summary

In this step we installed PowerShell, the Git client, and downloaded the LazyStack management scripts into an OrgMgmt folder.
