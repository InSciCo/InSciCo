SMF 4.2 Configure Developer Workstation

This step is performed on the developer's workstation. The previous step produced a file like "OrgDevJoe_wecome.txt". Send this file ot the developer and have them follow the steps in Configure Developer Workstation.

If you are creating a developer's account on the Management Workstation you can also use this process to configure the Management Workstation as a developer's workstation.
