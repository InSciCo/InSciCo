SMF 4.1 Create GitHub Repository

Prerequisites: Create GitHub Account (1.4) complete
Estimated Time: 2 minutes
Estimated Cost: none

See this newtab}}> Creating a new repository documentation on GitHub.

Note: The new repository must be accessible from the "Personal account" used to generate the "Personal access token" in "Create GitHub Account" (step 1.4).

  1. Log in to GitHub.
  2. Create the repository if you don't already have one for this stack.
  3. Navigate to the Code tab for the repository.
  4. Click on the "Code" button. The download options drop-down menu appears.
  5. Under the HTTPS option you will see the url for your repository.
  6. Click the clipboard option next to the url to copy that url.
  7. Make a note of this url as you will need it in the following steps.
