SMF Step 4 - Configure GitHub Repository

Prerequisites: Create System (Step 3) complete
Estimated Time: 2 minutes
Estimated Cost: none
Perform this step on the Management Workstation.

In this step you configure one or more GitHub Repositories for your system. Each repository defines a serverless stack.

  • 4.1 Create GitHub Repository for Serverless Stack
  • 4.2 Configure System Test Account for Continuous Integration and Continuous Deployment
  • 4.3 Configure System Production Account for Continuous Deployment

Note: If you do not yet have a stack to define you can stop the SMF setup here and come back to this step later. For instance, the LazySTack SMF Tutorial will direct you to this step when creating a new serverless stack.
