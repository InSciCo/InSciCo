AWS Production Account Setup

Prerequisites: AWS Org. Setup Part 1 complete.
Estimated Time: 2 minutes, Costs: none
Perform this step on the Management Workstation

  1. Open a PowerShell Terminal.
  2. CD into the OrgMgmt folder. Replace "Org" with your OrgCode.
  3. Execute the CreateProdAccount.ps1 script: 
    . .\CreateProdAccount.ps1

Step Summary

In this step we used the CreateProdAccount.ps1 script to create a Production Account under the Production Organization Unit. We added a Administrators group to the account with AdministratorAccess permissions. We added an IAM User for the account and added that user to the Administrators group. We created a AWS CLI profile allowing the Management Account IAM User to manage the Prod account with the OrganizationAccountAccessRole policy.
