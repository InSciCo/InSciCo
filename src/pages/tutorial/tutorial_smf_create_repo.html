Create Stack Repository

In the LazyStack Tools Tutorial you created a PetStore Solution containing a robust serverless stack. You published that stack to your sandbox AWS Account and accessed that stack using a PetStore client app. In this step we are going to create a GitHub repository containing your work so it can be shared with others and later on be used to drive a CI/CD process.

Visual Studio IDE (Windows) Steps
  1. Open the PetStore Solution in Visual Studio.
  2. Select the Visual Studio Git → "Create Git Repository" menu item. The Create a Git Repository dialog appears.
  3. Select your GitHub Organization in the "Owner" drop down combo. If no GitHub Organization exists, then just use your GitHub Account.
  4. Edit "Repository name" to a unique value. ex: "JoePetStore".
  5. Click on the "Create and Push" button. The following actions are taken:
    • A .git folder is added in the Solution root folder.
    • A .gitattributes file is added in the Solution root folder.
    • A .gitignore file is added in the Solution root folder.
    • The shared repository is added to GitHub.
