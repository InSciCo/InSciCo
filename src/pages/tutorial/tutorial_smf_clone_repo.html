1 Clone Repository

In this step we will clone your GitHub repository to your local system. The repository folder created will have only the following content:

  • .git folder - this folder contains the local git repository
  • .gitignore file - contains the ignore configuration for VisualStudio solutions
  • .gitattributes file
  • README.md - readme template for solution
  • LICENSE.txt - license file for solution
  1. Open a command terminal.
  2. Issue the following command: 
    
git clone https://github.com/InSciCo/OrgJoePetStore.git
    This creates a OrgJoePetStore folder configured with a local .git repository and connectivity to your GitHub Organization's OrgJoePetStore repository. Replace the git repo URL with the one provided with your credentials.
