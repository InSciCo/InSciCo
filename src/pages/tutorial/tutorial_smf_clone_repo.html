Clone Repository

This step must be performed on a Developer Workstation.

  1. Open a command terminal.
  2. Issue the following command: 
    
git clone https://github.com/InSciCo/OrgJoePetStore.git
    This creates a OrgJoePetStore folder configured with a local .git repository and connectivity to your GitHub Organization's OrgJoePetStore repository.
    Replace the git repo URL with the one provided by your ? (manager, what?)
  3. Copy the contents of your PetStore tutorial folder into the OrgJoePetStore folder.
  4. CD into the OrgJoePetStore folder.
