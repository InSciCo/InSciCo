1) Clone Repository

In this step we will clone your GitHub PetStore repository to your local system.

  1. Open a command terminal.
  2. Issue the following command (replacing GitHubOrg with your organizations GitHub Organization account): 
    git clone https://github.com/GitHubOrg/PetStore.git PetStoreSMF
    This creates a PetStoreSMF folder containing a clone of your GitHub Organization's PetStore repository. We named the folder PetStoreSMF so it wouldn't conflict with any PetStore folder you created in the LazyStack Tools Tutorial.
