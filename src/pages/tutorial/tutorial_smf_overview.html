Software Management Framework Tutorial

Prerequisites:
- SMF Install Steps 1 and 2 complete
- Developer Account Created
- Configure Developer Workstation Complete (using Developer Account information)
Estimated Time: 5 minutes
Estimated Cost: none
Some steps in this tutorial require access to Management Workstation
Some steps in this tutorial require access to Developer Workstation.

In this system we are going to create a new System and add a stack to that system. To make things simple we are going to use the PetStore stack developed in the LazyStack Tutorial. During the SMF installation process, a Management Workstation was configured with access to the AWS Management account for an AWS Organization. In addition, individual "Developer AWS Accounts" were created for each developer.

This tutorial requires access to the management workstation and developer workstation. If your workstation is not configured as both a management and developer workstation, then you will need to coordinate some steps in this tutorial with a team member that has access to a management workstation.

  • 1) Create a System - "Tut"
  • 2) Create PetStore repository on GitHub
  • 3) Install CI/CD
  • 4) Start Feature Development
  • 5) Create Pull Request
  • 6) Update Pull Request
  • 7) Merge Pull Request
  • 8) Checkout Latest Main
