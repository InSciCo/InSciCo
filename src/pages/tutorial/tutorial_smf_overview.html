Software Management Framework Tutorial

Prerequisites:
- SMF Install Steps 1 and 2 complete
- Developer Account Created
- Configure Developer Workstation Complete (using Developer Account information)
Estimated Time: 5 minutes
Estimated Cost: none
This tutorial is performed on the Developer Workstation.

In this tutorial we are going to add a stack to the Tutorial system created during Install Configure LazyStack SMF process. We assume you have completed one of the LazyStack Tools Tutorials, so we will use that local .NET solution to create the system stack. During the SMF installation process, individual AWS Accounts were created for each developer. You will need the credentials produced by that process to execute this tutorial. You may have already performed the Install Configure Developer Workstation process with those credentials, if not please update your Developer Workstation configuration using those credentials.

In addition, a GitHub Repository should have been created for you during the Install Configure LazyStack SMF process. The suggested naming of that repository was "OrgDevJoePetStore" where OrgDevJoe is the name of your developer account.

  • 2) Start Feature Development
  • 3) Create Pull Request
  • 4) Update Pull Request
  • 5) Merge Pull Request
  • 6) Checkout Latest Main
