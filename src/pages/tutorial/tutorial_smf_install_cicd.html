Install CI/CD

In this step we are going to install Continuous Integration and Continuous Deployment processes in the Tutorial Test Account and Tutorial Production Account.

Steps
  1. Add Release Stack to LazyStack.yaml
    1. Open the LazyStack.yaml file.
    2. Insert the following Stacks entry: 
      
Stacks:
  Rel:
    ProfileName: YourProfileName
    RegionName: us-east-1
    StackName: PetStore
    Stage: Rel 
    UriCodeTarget: Release/netcoreapp3.1
    Replace ProfileName and RegionName with your values.
  2. Save your edit.
