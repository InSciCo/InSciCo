2 Start Feature Development

  1. CD into the repository folder.
  2. Issue the following Git commands to create a new feature branch called OrgJoeF1 (replace OrgJoe with your OrgCode and Developer handle): 
    git branch OrgJoeF1
  3. Copy the contents of your PetStore tutorial folder into the OrgJoePetStore folder. If your PetStore tutorial folder contains a .git folder, .gitignore file, or .gitattributes file do not overwrite the one already present in the orgJoePetStore folder. It's OK to overwrite the README.md and LICENSE.txt files.
  4. Stage, commit and push feature content to GitHub:
      Visual Studio Git Integration
      Command Line
      1. Issue the following Git commands to stage, comment and push to the main branch: 
         git add --all
         git commit -m "initial commit"
        git push -u origin OrgJoeF1
  5. Get AWS Settings to call the OrgJoeF1 stack.
  6. Test the OrgJoeF1 stack.
