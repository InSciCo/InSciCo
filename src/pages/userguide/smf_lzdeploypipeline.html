LzDeployPipeline Script

The LzDeployPipeline.ps1 PowerShell script is run on the Management Workstation to deploy a specific CodeBuild project. This is required after you make changes to

IAMUserCredsPolicy.json

This policy grants the developer those permissions typically required to publish Serverless Stacks to their AWS Sandbox Account. Use the LzUpdateDevCreds.ps1 script to update the policy for an existing developer's account when you make changes to the local IAMUserCredsPolicy.json document.

Running the LzUpdateDevCreds Script
  1. Open a PowerShell terminal

  2. CD into your LazyStackSettings folder.

  3. ..\LazyStackSMF\LzUpdateDevCreds.ps1
