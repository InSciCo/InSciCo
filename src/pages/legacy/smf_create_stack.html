SMF Step 3 - Configure GitHub Repository

Prerequisites: Create System (Step 2) complete
Estimated Time: 2 minutes
Estimated Cost: none
Perform this step on the Management Workstation.

In this step you configure one or more GitHub Repositories for your system. Each repository defines a serverless stack. We suggest you configure one tutorial repository for each developer who will be performing the LazyStack Tutorial. The following steps will demonstrate this by creating a serverless stack repo for the "OrgDevJoe" developer you created earlier.

  • 3.1 Create GitHub Repository for Serverless Stack
  • 3.2 Configure System Test Account for Continuous Integration and Continuous Deployment
  • 3.3 Configure System Production Account for Continuous Deployment
