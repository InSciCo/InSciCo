Software Management Framework (SMF)

Managing software development, testing, deployment and maintenance is critical to your business success. LazyStack provides a complete end-to-end software development management tool-chain tailored to and integrated seamlessly with AWS and GitHub. GitHub is a very popular software version control repository service.

The LazyStack SMF helps ensure your team is collaborating effectively on delivering new features and code fixes to your customers as quickly and efficiently as possible. The LazyStack SMF provides the tools, templates, scripts, practices, and examples necessary to implement the tool-chain quickly and with confidence that it complies with industry best-practices.

A Software Management Framework you can Grow

The LazyStack SMF allows you to start with a simple systems comprised of a single serverless stack and then easily grow as needed to develop more complex systems.

  • AWS Organization
    AWS Organizations aggregate AWS Billing and allow you to easily apply various usage limits and operational policies across all AWS Accounts in that organization.
    • One or more Developers in Organization
    • One or more Systems in an Organization
      Systems are used to isolate resources like databases by line-of-business, product, or customer.
      • One or more Serverless Stacks in a System
        Multiple serverless stacks separate sub-systems. For instance you might have a Consumer stack that services a consumer mobile app and a Backoffice stack that services a desktop application that employees use.
