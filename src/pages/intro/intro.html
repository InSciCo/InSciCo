LazyStack Executive Overview

LazyStack's mission is to enable technical entrepreneurs to quickly field mobile and web applications leveraging scalable cloud-based resources at the lowest initial cost, without sacrificing security, scalability or features. LazyStack leverages the AWS Serverless Application Model (SAM) to enable you to build and deploy "application stacks" in the AWS cloud.

It is best practice to create an API specification that defines all the service features available on your AWS Stack. Your client application uses this specification to implement calls to the AWS Stack. The amount of "plumbing" code and AWS stack configuration necessary to implement a feature rich API specification is often quite large. Plumbing code connects the client app and AWS Lambda function implementing a stack feature. Plumbing code is repetitive and "boring" to write, making it an excellent candidate for code generation. In addition, writing complete and correct "SAM" templates used to deploy a stack on AWS is tedious and requires an intimate knowledge of AWS resources.

LazyStack eliminates labor intensive and error prone coding and template development so see real business benefits:

  • Clear Requirements
    • An API specification reduces misunderstandings and wasted implementation cycles.
    • API specifications can be versioned, making it simple to manage incremental improvement in a product.
    • API specifications make it easy to partner with other parties.
    • API specifications make it easier to explain technical concepts to business stakeholders.
    • Tools exist to help author effective API specifications.

  • Lower Cost Development
    • Expensive hand-coding of "boring" plumbing code is eliminated.
    • Innovation is easier because new plumbing code and templates are automatically generated when the API specification changes.
    • Generated code is consistent, bug-free and reliably compliant with best-practices.
    • Developers are freed up to work on features that can't be generated.
    • Generated code is far less expensive to test and verify.
    • LazyStack publishes a number of open-source MIT licensed libraries to kick-start your app development.

  • Lower Cost DevOps
    • Leverages AWS Serverless Application Model - generates AWS SAM templates from API Specification.
    • Generated templates demonstrate correct configuration of AWS resources.
    • Generated templates are consistent, bug-free and reliably compliant to best-practices.
    • Generated templates can be easily extended with hand-coded resource specifications.
    • Manage Dev, Test and Prod stacks with simple Software Development Life Cycle (SDLC).
    • LazyStack easily integrates with advanced SDLC like CI services as business grows.

LazyStack makes innovation faster and less expensive by reducing the costs associated with code and stack configuration changes. LazyStack accelerates your time-to-market so you start monetizing your app earlier. Leveraging the AWS Serverless Application Model ensures your app has the potential to scale reliably and quickly and launch at low cost using AWS free-tier services.

What is LazyStack?

LazyStack ingests an OpenApi (swagger) REST API specification and generates the AWS Server Application Model (SAM) configuration and C# projects necessary to stand up a fully featured commercial-grade application stack in AWS. It also generates a client application SDK making it simple to call that stack from C# applications.

LazyStack is comprised of a free stack generator tool and some MIT licensed open-source support libraries.

  • LazyStack Generator - Free Visual Studio IDE Extension, and Free DOTNET CLI Tool
  • LazyStackAuth - MIT Licensed authentication library
  • LazyStackDynamoDBRepo - MIT Licensed CRUDL library for DynamoDB

LazyStack builds on these cornerstone technologies:

  • OpenApi (swagger) REST API specification
  • NSwag -- .NET Code Generation
  • Microsoft .NET Core & C#
  • .NET Complier Platform SDK
  • AWS Serverless Application Model (SAM)
