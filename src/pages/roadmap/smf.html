LazyStack Software Management Framework Version 1.0.x

LazyStack SMF is now depreciated in favor of the LazyStack SaaS Platform. No further updates for SMF are planned. However, The SMF project is a great example of using the AWS CLI and PowerShell scripting to build AWS Organizations, Accounts and perform CI/CD integration with GitHub and AWS CodeBuild.
