PetStore Serverless Stack Deployment

The PetStore stack is a "Pure Serverless Stack" where only AWS Serverless Resources are used. AWS Serverless Resources offer on-demand, auto-scaling configurations and do not require creating a network management layers with a Virtual Private Cloud.

The AWS Resources included in the pure serverless stack example include:

  • AWS::Cognito::UserPool
  • AWS::Cognito::UserPoolClient
  • AWS::Cognito::IdentityPool
  • AWS::Cognito::IdentityPoolRoleAttachment
  • AWS::IAM::Role
  • AWS::Serverless::Api
  • AWS::DynamoDB::Table
  • AWS::Serverless::Function
