Lambda Function Resources and Projects

Defining a tag in the OpenApi specification generates a Lambda Function resource entry in the serverless.template file, a Lambda project and a Controller project.

Note: The tag must be assigned to a OpenApi route (Path + Operation) for the resource entry to be generated.
