LazyStack Generator Tools

LazyStack ingests your application's OpenAPI specification to generate your C# projects and Serverless Application Model (SAM) file(s). In addition, LazyStack reads generator "directives" from your application's LazyStack.yaml file to control various generation options.

There are two LazyStack tools:

  • Generate Projects
  • Generate AWS Settings File
  • Generate Projects
  • Stacks Directive
  • DefaultApi Directive
  • ApiTagMap Directive
  • AwsResources Directive
  • ProjectOptions Directive
  • AwsTemplate Directive
  • Escaped Characters
  • Resource Merge Priority
  • Generate AWS Settings
