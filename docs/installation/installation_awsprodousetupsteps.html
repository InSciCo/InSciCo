AWS Review Prod Organizational Unit Setup

Prerequisites:
- Test Account Setup Complete
Estimated Time: 2 minutes
Estimated Cost: none
Perform this step on the Management Workstation.
  1. Open a PowerShell Terminal.
  2. CD into the OrgMgmt folder. Using your OrgCode in place of "Org".
  3. Execute the Prod_OU_Create.ps1 script: 
    . .\Prod_OU_Create.ps1
    Here is a script run with sample inputs: 
    
 Prod_OU_Create.ps1 - V1.0.0
 This script creates an Organizational Unit for Production stacks.
 Note: Press return to accept a default value.
 Enter your OrgCode (ex: Lz): Az
 Enter your AWS CLI Management Account Profile (default: AzMgmt):
 Please Review and Confirm the following:
     OrgCode: Az
     Management Account Profile: AzMgmt
     Organizational Unit to be created: AzProdOU
 Confirm y/n: y 
 Processing Starting
 Creating Organizational Unit:
    - AzProdOU created
 Processing Complete
Step Summary

In this step we used the Prod_OU_Create.ps1 script to create a Prod Organizational Unit under the Organization.
