Developer Account Setup

Prerequisites: Organization Setup complete
Estimated Time: 2 minutes, Costs: none
Perform this step on the Management Workstation

In this step we will run the CreateDevAccount script to create a new Developer Member Account belonging to the AWS organization and move this account into the OrgDevOU Organizational Unit.

  1. Open a PowerShell Terminal.
  2. CD into the OrgMgmt folder. Using your OrgCode in place of "Org".
  3. Execute the CreateDevAccount.ps1 script: 
    . .\CreateDevAcct.ps1
    Here is a script run with sample inputs:

Step Summary

In this step we used the CreateDevAcct.ps1 script to create a new Dev Account and move that account into the Dev Organizational Unit. The script also created a Developers group and in the new Dev Account, created a new IAM User for the developer and assigned that IAM User to the Developers group. The Developers Group contains a policy allow the developer to publish up to three Serverless Stacks with the pre-defined names SS1, SS2, SS3.

The pre-defined Serverless Stack names are required to associate necessary IAM permissions to the Developers IAM User account to successfully publish Serverless Stacks.
