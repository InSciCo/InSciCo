SMF 2.1 Create Developer Account

Prerequisites:
- Organization Setup complete
- Unique Email address (or alias) to use for Developer Account
Estimated Time: 2 minutes
Estimated Cost: none
Perform this step on the Management Workstation.

In this step we will run the Dev_Account_Create.ps1 script to create a new Developer Member Account belonging to the AWS organization and move this account into the OrgDevOU Organizational Unit. Execute these steps on your Management Workstation.

  1. Open a PowerShell terminal.
  2. CD into the OrgMgmt folder. Replace "Org" with your OrgCode.
  3. Execute the Dev_Account_Create.ps1 script: 
    . .\Dev_Account_Create.ps1
    Here is a script run with sample inputs: 
    
Dev_Account_Create.ps1 - V1.0.0
This script adds a developer account to the Dev Organizational Unit.
Note: Press return to accept a default value.
Enter your OrgCode (ex: Lz): Az
Enter your AWS CLI Management Account Profile (default: AzMgmt):
Enter Region (default us-east-1):
Enter the Development OU Name (default: AzDevOU):
Enter the Developer's Handle (ex: Joe): Joe
Enter the Developer's Account Name (default: AzDevJoe): 
Enter the Developer's IAM User Name (default: AzDevJoe):
Note: An email address can only be associated with one AWS Account. 
Enter an Email Address for the new account's Root User: me+AzDevJoe@gmail.com
Please Review and Confirm the following:
    OrgCode: Az
    Management Account Profile: AzMgmt
    Development OU: AzDevOU
    Development Account to be created: AzDevJoe
    Development Account IAM User Name: AzDevJoe
    Development Organization Unit: AzDevOU
    Email Address for Account's Root User: me+AzDevJoe@gmail.com 
Confirm y/n: y 
Processing Starting
Creating Developer Account AzDevJoe 
    - Checking for successful account creation. TryCount=1 
    - AzDevJoe account creation successful. AccountId: 123456789123
Adding AzDevJoeAccess profile and associating it with the AzMgmt profile 
Creating Developers Group in AzDevJoe account 
    - Adding policy PowerUsersAccess 
    - Adding policy IAMUserChangePassword
Creating IAM User AzDevJoe in the AzDevJoe account.
    - Adding the IAM User AzDevJoe to the Developers group in the AzDevJoe account.
    - Writing the IAM User Creds into AzDevJoe_welcome.txt
Processing Complete
Send the AzDevJoe_welcome.txt file to the Developer or use it yourself if you are also that developer.
The file contains the URL to login to the AWS Account AzDevJoe and the initial password (password reset

The welcome.txt file has content that looks something like this: 


    AWS Account Console: https://123456789012.signin.aws.amazon.com/console
    IAM User: AzDevJoe
    Temporary Password: 93939aA!
    Please login, you will required to reset your password.
    Please login as soon as possible.
    Follow the "Dev profile Setup" instructions at http://lazystack.io/installation/installation_awsdevprofilesteps.html

Step Summary

In this step we used the Dev_Account_Create.ps1 script to create a new Dev Account and move that account into the Dev Organizational Unit. The script also created a Developers group and in the new Dev Account, created a new IAM User for the developer and assigned that IAM User to the Developers group.
