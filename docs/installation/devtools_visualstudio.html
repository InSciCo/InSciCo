Dev WS Step 3.1 LazyStack for Visual Studio Windows

Install Steps

Prerequisites: AWS Account, AWS CLI, AWS SAM CLI.

Recommended: .NET CLI, LazyStack .NET CLI tool.

  1. Install/Update Visual Studio 2019 for Windows
      Required Workloads:
    • ASP.NET and web development
    • .NET desktop development
    • Mobile development with .NET
    • .NET Core cross-platform development (.NET 5)
  2. Install Visual Studio Extensions
    1. Open Visual Studio for Windows
    2. Select Extensions → Manage Extensions. The Manage Extensions Dialog appears.
    3. Type "AWS Toolkit" into the search field. The "AWS Toolkit for Visual Studio 2017 and 2019" extension is listed.
    4. Click the Download button. The extension is scheduled for install.
    5. Select Extensions → Manage Extensions. The Manage Extensions Dialog appears.
    6. Type LazyStack into the search field. The LazyStackVsExt extension is listed.
    7. Click the Download button. The extension is scheduled for install.
    8. Close and reopen Visual Studio. The extensions will be installed.

Updating the LazyStack and AWS Toolkit Extensions

  1. Open Visual Studio for Windows
  2. Select Extensions → Manage Extensions. The Manage Extensions Dialog appears.
  3. Click "Updates" in the Manage Extensions navigation tree.
  4. Select extensions for update (if any have pending updates).
  5. Close and reopen Visual Studio for the extension(s) to be fully updated.
