2 Start Feature Development

  1. CD into the PetStoreSMF repository folder.
  2. Issue the following git commands to create a new feature branch called OrgJoeF1 (replace OrgJoe with your OrgCode and Developer handle): 
    git branch OrgJoeF1
  3. Stage, commit and push feature content to GitHub:
      Visual Studio Git Integration
      Command Line
      1. Issue the following Git commands to stage, comment and push to the main branch: 
         git add --all
         git commit -m "initial commit"
        git push -u origin OrgJoeF1
  4. Get AWS Settings to call the OrgJoeF1 stack.
  5. Test the OrgJoeF1 stack.
